================================================================================
USAGE EXAMPLE
================================================================================

Create and submit a purchase order:

  const result = await createWorkflowDocument('PURCHASE_ORDER', {
    vendorName: 'Acme Corp',
    vendorId: 'V-001',
    items: [...],
    totalAmount: 10000,
    currency: 'ZMW',
    deliveryDate: new Date(),
    specialInstructions: 'Urgent'
  })

  if (result.success) {
    await submitDocument(result.data.id)
  }

Get pending approvals for your role:

  const result = await getPendingApprovals('FINANCE_OFFICER')

Approve a document:

  const result = await approveDocument(documentId, 'Approved')
  // Document automatically advances to next stage!
