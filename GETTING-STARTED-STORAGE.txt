🚀 QUICK START
═════════════════════════════════════════════════════════════════════════════

1. VIEW PURCHASE ORDERS
   Navigate to: http://localhost:3000/purchase-orders
   You should see 5 purchase orders with different statuses

2. USE IN A COMPONENT
   import { usePurchaseOrdersQuery } from '@/hooks/use-storage-queries';

   function MyComponent() {
     const { data: orders } = usePurchaseOrdersQuery();
     return <OrderList orders={orders} />;
   }

3. USE STORAGE HOOKS DIRECTLY
   import { getPurchaseOrders, savePurchaseOrder } from '@/lib/storage';

   const orders = getPurchaseOrders();
   const saved = savePurchaseOrder(updatedOrder);

4. DEBUG IN BROWSER CONSOLE
   // View all data
   JSON.parse(localStorage.getItem('liyali_purchase_orders'))

   // Get statistics
   import { getStorageStats } from '@/lib/storage';
   getStorageStats()
